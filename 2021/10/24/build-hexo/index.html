build hexo

部署blog

Hexo搭建步骤
https://hexo.io/zh-cn/docs/
安装Git
https://gitforwindows.org/ 下载安装

安装完查看git版本：$git –version

安装Node.js
https://nodejs.org/en/download/ 选择LTS的window版本，下载安装
安装完查看node.js版本
$node -v
$npm -v

更改npm镜像源：npm config set registry https://registry.npm.taobao.org/
查看npm镜像源：npm get registry https://registry.npm.taobao.org/

安装Hexo
$npm install -g hexo-cli

建站

初始化博客目录
安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。
$ hexo init
$ cd
$ npm install

命令：
清理缓存： hexo clean
生成静态文件： hexo g/generate
生成静态文件： hexo s/server
发布到github：hexo d
hexo clean && hexo g && hexo s
本地访问地址：http://localhost:4000

配置
_config.yml 网站的 配置 信息，可以在此配置大部分的参数。
https://hexo.io/zh-cn/docs/configuration

生成SSH添加到GitHub
生成key，可以git部署网站
ssh-keygen -t rsa -b 4096 -C “your_email@example.com“
然后需要配置
git config –global user.email “you@example.com”
git config –global user.name “Your Name”
将这个文件拷贝到git的https://github.com/settings/keys
ssh -T git@github.com查看是否配置成功

GitHub创建个人仓库
新建一个 repository。如果你希望你的站点能通过域名 <你的 GitHub 用户名>.github.io 访问，你的 repository 应该直接命名为 <你的 GitHub 用户名>.github.io。

将hexo部署到GitHub

私有 Repository
安装 hexo-deployer-git.：
$ npm install hexo-deployer-git –save

在 _config.yml（如果有已存在的请删除）添加如下配置：
deploy:
type: git
repo: https://github.com//
运行 hexo
clean && hexo deploy 。
查看 username.github.io 上的网页是否部署成功。

发布文章
hexo new post “title”
然后用编辑器修改好文本，发布
hexo clean && hexo g && hexo s
hexo clean && hexo deploy
